# **ğŸ“– Overview**

SCAI (Strategic Commerce AI) is not just a chatbot; it is a goal-oriented autonomous sales agent engineered to maximize B2B revenue.

Unlike passive support bots, SCAI actively manages the sales lifecycle. It connects a Large Language Model (Llama 3.1) directly to your Enterprise Resource Planning (ERP) system via a Spring Boot backend. It negotiates prices using real-time math proofs, manages inventory states, and executes aggressive upsell strategies based on order volume.

# **ğŸ›  Installation & Setup**

Prerequisites

- Java 17+ & Maven

- Python 3.11+ (Anaconda recommended)

- MySQL Database

- Ollama (with llama3.1 model pulled)

1. Backend Initialization (Java)

Navigate to the backend directory and start the server:

    cd backend-java
    mvn spring-boot:run


Server will initialize on port `8080`.

2. AI Environment Setup (Python)

Install the required dependencies:

    cd agent-python
    pip install langchain langchain-community langchain-ollama langchain-chroma requests python-dotenv


3. Data Ingestion (ETL)

Sync your SQL inventory with the Vector Database:

    python indexer.py


4. Launch SCAI Agent

Start the interactive sales terminal:

    python agent-python.py
